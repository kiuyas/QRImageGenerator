QR Image Generator
====

## Description

Generate QR code image using [qrcode](https://www.npmjs.com/package/qrcode).

Node.js programming.

## How to use

1. Clone this folder.
2. npm install
3. node app [data you want generate] [output filename]
  - ex.) node app "this is a pen." "qr.png"
